# TerminalVibes — Image Generation Prompts

Every image is a **cinematic, whimsical teaching poster**: a beautiful storybook
world that carries real lesson information (exact commands, labeled diagrams,
legends). Wonder AND payload — never just decoration, never sterile.
Each prompt below is **fully self-contained** — copy one block, paste it into
your generator, done. Nothing refers to a shared style guide.

**Pipeline:** generate banners at 16:9 (the house size is 1672×941 — resize
oversized generations before wiring). Drop PNGs into `static/images/` with the
exact filename, then run `node scripts/optimize-images.mjs` (PNG → WebP,
deletes originals), then `node scripts/make-poster.mjs` to refresh the README
collage. `node scripts/make-playground-shot.mjs <baseUrl>` refreshes the README
playground screenshot. The og-image stays a **1200×630 PNG** (`static/og-image.png`).

**Proofread the rendered text.** These prompts specify exact strings; modern
generators render them well but occasionally typo — regenerate rather than
ship a misspelled command. Keep quiet margins; the site never overlays text on
banners, but breathing room is part of the look.

**Not generated:** `static/images/thompson-ritchie.jpg` is a real public-domain
photograph of Ken Thompson and Dennis Ritchie at the PDP-11 (source it from
Wikimedia Commons), used in the history section with a credit caption.
